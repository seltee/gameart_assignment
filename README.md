# GameArt assigment done by Dmitrii Shashkov

### Core libs used
PixiJS 8

### Steps to run this project:
1. ```yarn```
2. ```yarn dev```
3. Visit http://localhost:8080/
4. Use /?forceWin to win the game or /?forceLose to lose the game

### Building the game
1. ```yarn```
2. ```yarn build```
3. check build folder
